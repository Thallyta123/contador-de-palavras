Exercício: Complete as funções JavaScript

Seu objetivo é implementar as seguintes funcionalidades no arquivo JavaScript:

  • contarPalavras(texto): Retorna o número de palavras no texto
  • contarCaracteres(texto): Retorna o número de caracteres (incluindo espaços)
  • palavraMaisFrequente(texto): Retorna a palavra que mais aparece no texto
  • tempoLeitura(texto): Calcula o tempo estimado de leitura (considerando 200 palavras por minuto)
  • calcularLegibilidade(texto): Retorna "Fácil", "Médio" ou "Difícil" baseado no comprimento médio das palavras
  • gerarNuvemPalavras(texto): Gera uma nuvem de palavras com as 10 palavras mais frequentes

Nota: As funções já estão implementadas no arquivo script.js, mas você pode modificar e melhorá-las.
